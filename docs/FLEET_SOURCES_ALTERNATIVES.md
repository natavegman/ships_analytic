# Альтернативные источники данных о судах и судовладельцах

Сведения о судовладельце в российских открытых реестрах (РМРС, FleetPhoto и др.) **скрыты примерно с 2022 года в связи с санкциями**. Ниже — альтернативные достоверные источники, где можно получить или сопоставить данные о судах и владельцах/операторах.

---

## 1. Цербер (Россельхознадзор) — основной источник в проекте

- **URL:** https://cerberus.vetrf.ru/cerberus/certified/pub  
- **Что даёт:** реестр предприятий и объектов, сертифицированных для экспорта (в т.ч. рыба и морепродукты). Есть **ИНН хоз. субъекта** и вид объекта «предприятия (суда) по добыче…» — по сути, связка **судно ↔ компания (ИНН)**.  
- **Как используем:** выгрузка XLS → `data/cerberus_export.csv` → слияние с `company_groups_enriched` по ИНН → колонки Цербер_судов, Цербер_страны в `output/companies_with_export.csv`.  
- Подробнее: [CERBERUS_EXPORT_AND_FLEET.md](CERBERUS_EXPORT_AND_FLEET.md).

---

## 2. FAO Global Record of Fishing Vessels

- **URL:** https://globalrecord.fao.org/ (поиск: https://globalrecord.fao.org/advanced-search)  
- **Кто ведёт:** ФАО ООН.  
- **Что даёт:** глобальный реестр промысловых, рефрижераторных и вспомогательных судов; в расширенном поиске есть фильтр **Owner / Operator** и исторические данные. Поиск по IMO, названию, флагу, типу судна.  
- **Ограничения:** публичного API или массовой выгрузки в открытом доступе нет; поиск через веб-интерфейс. Для выгрузок/API можно уточнить у ФАО (Contact us на сайте).  
- **Идея для проекта:** по IMO или названию судна из Цербера искать судно в Global Record и при наличии — брать Owner/Operator для ручной или полуавтоматической сверки с нашими компаниями.

---

## 3. Global Fishing Watch (GFW)

- **URL:** https://globalfishingwatch.org/  
- **Документация API:** https://globalfishingwatch.org/our-apis/documentation  
- **Кто ведёт:** некоммерческая организация (партнёрство Oceana, SkyTruth, Google).  
- **Что даёт:**  
  - **Vessels API** — поиск судов, получение судна по id, в т.ч. **vessel identity** (идентификация, в т.ч. данные об ownership/операторе, где доступны).  
  - Данные по активности (AIS, события, порты) и отчёты по регионам.  
- **Доступ:** API по токену (бесплатно для некоммерческого использования). Токен: https://globalfishingwatch.org/our-apis/tokens  
- **Идея для проекта:** по названию судна или IMO из Цербера вызывать GFW Vessels API (search / get by id), получать идентификацию и при наличии — владельца/оператора, затем сопоставлять с ИНН/названиями компаний в `company_groups_enriched`.
- **IMO в GFW:** в ответах API v3 (vessels/search и vessel by id) при наличии передаётся поле **IMO** (уникальный номер судна). В проекте IMO извлекается и сохраняется в `data/gfw_our_vessels.json` (поле `imo`). Если в GFW IMO нет — см. Lloyd's List ниже.

## 4. Lloyd's List Intelligence / Lloyd's Register

- **Назначение:** глобальные базы по судам (Lloyd's List Intelligence, Lloyd's Register — коммерческие продукты).
- **Что даёт:** уникальный **IMO** для судна, технические данные, история владельцев/операторов. IMO не меняется в течение жизни судна и подходит как ключ для сопоставления между источниками (Цербер, GFW, FleetPhoto, РМРС).
- **Ограничения:** платный доступ, API/выгрузки по договору. Для бесплатной альтернативы по IMO можно использовать поиск по **IMO в GFW** (если судно есть в GFW), **FleetPhoto** (на карточке судна указан IMO в таблице), **FAO Global Record** (поиск по IMO), **IMO GISIS** (ручной поиск по IMO Number).

## 5. ГСС / ИСРС (Росморречфлот, marinet.ru)

- **ГСС (Глобальный справочник судов):** https://gss.marinet.ru  
- **ИСРС (Реестр судов):** https://srs.marinet.ru  
- **Кто ведёт:** ФГБУ «ИАСЦ» Росморречфлота.  
- **Что заявлено:** учёт государственной регистрации судов и прав на них, в т.ч. сведения о судах и судовладельцах.  
- **Важно:** после 2022 года доступ к данным о судовладельцах мог быть ограничен (санкции). Стоит проверить текущий интерфейс ГСС/ИСРС — если судовладелец в открытом доступе есть, можно использовать для сверки; если нет — учитывать только как источник технических данных о судне (название, IMO, порт приписки и т.д.).  
- Контакты: marinet@marinet.ru, 8 (495) 626-98-19.

## 5. IMO GISIS (по компании, не по судну)

- **URL:** https://gisis.imo.org/ (модуль Ship and Company Particulars).  
- **Что даёт:** поиск по IMO Number, по IMO Company Number — данные о компании.  
- **Ограничения:** выгрузка/автоматический сбор данных через скрипты без согласования с оператором (S&P Global) не допускается; допустим ручной поиск по компаниям/судам.  
- Подходит для точечной проверки компаний по IMO Company Number, если он известен из других источников.

## Рекомендации по использованию в Quotas_analytic

| Задача | Источник |
|--------|----------|
| Связка судно ↔ компания (ИНН) по РФ | **Цербер** (уже интегрирован) |
| Доп. проверка/обогащение: владелец/оператор по названию или IMO | **FAO Global Record** (ручной поиск), **GFW Vessels API** (поиск по API при наличии токена) |
| Технические данные по судну (IMO, порт, тип) | Цербер, FleetPhoto/РМРС (без владельца), при наличии — ГСС |
| Точечная проверка компании по IMO | IMO GISIS (ручной поиск) |

При появлении открытого API или выгрузки с полем «судовладелец»/ИНН (например, от ФАО или после изменения политики ГСС/ИСРС) можно добавить в проект скрипт загрузки и слияния по ИНН/названию компании.
