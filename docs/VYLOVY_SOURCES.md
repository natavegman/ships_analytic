# Источники данных по выловам (Росрыболовство и др.)

Мониторинг работы флота ведёт **Федеральное агентство по рыболовству** (Росрыболовство). Ниже — обзор возможных источников, в т.ч. неочевидных, для загрузки информации по выловам в проект.

---

## 1. Официальные открытые данные (без API)

### 1.1 Публичного API по выловам нет

- На **fish.gov.ru** и в **Отраслевой системе мониторинга (ОСМ)** — [osm.gov.ru](https://osm.gov.ru/) — **публичного API для выловов не предусмотрено**. ОСМ предназначена для рыбопользователей и контролирующих органов, доступ через личный кабинет; судовые донесения и детальные выловы по судам в открытый доступ не выкладываются.

### 1.2 Портал открытых данных РФ

- **data.gov.ru** — единый портал открытых данных. Раздел Росрыболовства: [data.gov.ru (организации)](https://data.gov.ru/organizations/9ba53a27-8d66-4438-8ad1-c14f3b447a1e).
- На странице [fish.gov.ru/otkrytye-dannye](https://fish.gov.ru/otkrytoe-agentstvo/otkrytye-dannye/) в реестре наборов ОД перечислены в основном перечни (территориальные органы, планы проверок, вакансии и т.п.). **Отдельного датасета «выловы по судам/компаниям» в списке нет.**
- **Контакт по открытым данным:** Литовченко Павел Александрович — +7 (495) 987-05-13 доб. 0486, [litovchenko@fish.gov.ru](mailto:litovchenko@fish.gov.ru). Имеет смысл направить **запрос на публикацию** набора данных по освоению квот/выловам (например, по компаниям/ИНН или по бассейнам и видам ВБР в табличном виде CSV/XLSX).

## 2. Разделы fish.gov.ru с данными о вылове

### 2.1 Освоение рекомендованных объёмов вылова

- Страница: [Освоение рекомендованных объемов вылова](https://fish.gov.ru/otraslevaya-deyatelnost/organizacziya-rybolovstva/osvoenie-rekomendovannyh-obemov-vylova/).
- **Что есть:** сведения об объёмах добычи (вылова) ВБР, по которым ОДУ не устанавливается (внутренние воды, морские воды, по годам).
- **Форматы:** в основном **PDF** (своды 2019–2025). Для части периодов есть **XLS/XLSX по бассейнам** (например, Дальневосточный, Северный, Западный, Азово-Черноморский, Волжско-Каспийский по состоянию на 2016–2018).
- **Ограничение:** данные по **бассейнам и видам ВБР**, а не по судам или по ИНН/компаниям. Для привязки к нашим судам/компаниям можно использовать только косвенно (бассейн, объект лова).

### 2.2 Статистика добычи (вылова) ВБР

- Раздел: [Статистика и аналитика](https://fish.gov.ru/otraslevaya-deyatelnost/ekonomika-otrasli/statistika-i-analitika/).
- **Что есть:** «Сведения об улове рыбы и добыче других водных биоресурсов» — отчёты за периоды (январь–март, январь–июнь, январь–декабрь и т.д.) с 2012 по 2025 год.
- **Формат:** **PDF**. Внутри — таблицы с агрегированной статистикой по РФ и/или бассейнам, **не по судам и не по компаниям**.
- **Вариант использования:** парсинг PDF (извлечение таблиц) для получения рядов «период — бассейн — объект — объём». Для карточки судна пригодно только как контекст по бассейну/объекту, не как «вылов этого судна».

### 2.3 Сервер calculations.fish.gov.ru (проверено)

- **Корень:** [calculations.fish.gov.ru/](https://calculations.fish.gov.ru/) — каталоги по **годам**: 2019, 2020, …, 2026.
- **Внутри года** (например [2025/](https://calculations.fish.gov.ru/2025/)): папки по **бассейнам** — Дальневосточный, Северный, Западный, Волжско-Каспийский; в Дальневосточном есть подпапка «Международное рыболовство».
- **Внутри бассейна:** снимки «Перечни и расчеты по состоянию на ДД.ММ.ГГГГ» (например 20.12.2024, 28.12.2024, 18.04.2025). В снимке — подпапки:
  - **перечень приб** / **перечень пром-приб** — перечни по видам ВБР (прибрежное / промыслово-прибрежное);
  - **расчет** / **Расчеты** / **Перечни** — файлы **XLSX** по объектам: `0_1_Камбалы дальневосточные.xlsx`, `0_2_Минтай.xlsx`, `0_3_Минтай.xlsx`, `0_8_Сельдь тихоокеанская.xlsx`, крабы, креветки, трубачи, трепанг и т.д.
- **Содержимое файлов:** по названиям и структуре каталога это **перечни и расчёты распределения квот** (кто и сколько получает по виду ВБР), а не отчёты о фактическом вылове по судам/компаниям. Формат — XLSX, пригоден для парсинга; при необходимости можно скачать выборочные листы и проверить наличие колонок «освоено»/«вылов», но основное назначение сервера — **квоты и расчёты**, не оперативная статистика вылова.
- **Вывод:** для **выловов** calculations.fish.gov.ru не заменяет отчёты «Статистика и аналитика» и «Освоение рекомендованных объёмов» на основном сайте. Для **квот и привязки компаний к объектам ВБР** — потенциально полезен (парсинг XLSX по бассейнам и датам).

## 3. Отраслевая система мониторинга (ОСМ) и ЦСМС

- **ОСМ:** [osm.gov.ru](https://osm.gov.ru/) — отраслевая система мониторинга ВБР и контроля за промысловыми судами.
- **ФГБУ «Центр системы мониторинга рыболовства и связи» (ЦСМС):** [cfmc.ru](https://www.cfmc.ru/) — сбор данных о местоположении судов, судовых суточных донесений, отчётов организаций; предоставление данных о производственной деятельности судов.
- **Доступ:** через личный кабинет (для рыбопользователей/контроля). **Публичного API для выловов в открытой документации нет.** Даже при наличии доступа к кабинету выгрузка по API для внешних систем, как правило, не предусмотрена.

## 4. Государственный рыбохозяйственный реестр (ГРР)

- Упоминается на [fish.gov.ru](https://fish.gov.ru/otraslevaya-deyatelnost/organizacziya-rybolovstva/gosudarstvennyj-rybohozyajstvennyj-reestr/) как автоматизированная система с документированной информацией по рыбохозяйственной деятельности.
- На data.gov.ru и в реестре открытых данных Росрыболовства **отдельного датасета ГРР с выловами по судам/компаниям не указано**. Возможность появления такого набора имеет смысл уточнять у ответственного за ОД (Литовченко).

## 5. Нетривиальные варианты

| Вариант | Описание |
|--------|----------|
| **Запрос в Росрыболовство** | Официальный запрос на публикацию открытых данных: «освоение квот / вылов по компаниям (ИНН)» или «вылов по видам ВБР и бассейнам» в формате CSV/XLSX на data.gov.ru или на fish.gov.ru. Контакт: litovchenko@fish.gov.ru. |
| **Парсинг PDF и XLS с fish.gov.ru** | Скачивание отчётов из разделов «Статистика и аналитика» и «Освоение рекомендованных объёмов», парсинг таблиц (PDF → таблицы, XLS/XLSX → строки). Результат — агрегаты по бассейну/периоду/объекту, без привязки к судну; для карточки судна — только как «вылов по бассейну/объекту». |
| **calculations.fish.gov.ru** | Проверено: только перечни и расчёты квот по видам ВБР (XLSX), снимки по датам и бассейнам. Фактических выловов по судам/компаниям нет; при необходимости можно использовать для квот/объектов. |
| **Отраслевые/аналитические отчёты** | Своды от отраслевых союзов, аналитических центров или региональных управлений Росрыболовства иногда содержат разбивку по компаниям или флотам; как правило, PDF или закрытые форматы. Источник нестандартный и не гарантированный. |

## 6. Международные источники (по выловам РФ — ограничены)

- **FAO Global Record** — реестр судов и идентификация, не детальные выловы по странам/суднам в формате, пригодном для РФ.
- **Global Fishing Watch** — активность и усилия судов, не официальные объёмы вылова в тоннах.
- **ГСОИ (FAO)** — обмен информацией в рамках СМГП (порты, инспекции), не статистика вылова.

## 7. Рекомендации для проекта Quotas_analytic

1. **Краткосрочно:** оставить в карточке судна блок «Выловы» с заглушкой «источник в разработке»; при появлении любого машиночитаемого источника (CSV/XLSX/API) добавить скрипт загрузки и вывод в карточку.
2. **Направить запрос** в Росрыболовство (litovchenko@fish.gov.ru) с просьбой раскрыть в виде открытых данных набор по освоению квот/выловам (хотя бы по компаниям/ИНН или по бассейнам и видам ВБР в табличном виде).
3. **Опционально:** реализовать загрузку и парсинг XLS/XLSX с раздела «Освоение рекомендованных объёмов» (по бассейнам) и, при возможности, PDF из «Статистика и аналитика» — для отображения в интерфейсе контекста по бассейну/объекту (без привязки к конкретному судну).
4. **Долгосрочно:** следить за появлением новых наборов на [data.gov.ru](https://data.gov.ru/) по тематике «рыболовство», «вылов», «освоение квот».
